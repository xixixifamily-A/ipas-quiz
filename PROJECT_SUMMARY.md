## 三、v3 出題風格（IPAS 114年考古題）

經分析三份 IPAS 114年第二季考古題 PDF，歸納出 8 項出題特徵：

### 風格比例
- **情境應用題 ~40%**：商業/技術場景（電商推薦、醫療診斷、金融風控、自駕車等）
- **「下列何者正確/不正確」~30%**：一題測四個知識點
- **直接問答 ~30%**：簡潔問答型
- **Python程式碼判斷題**：每科目至少 1~2 題
- **計算題**：精確率/召回率/F1、Z-score、IQR、變異係數等

### 新技術趨勢（相較舊版新增）
- AutoML 與自動化機器學習
- 模型壓縮（蒸餾、量化、剪枝）
- GDPR / 差分隱私
- SHAP / LIME 可解釋 AI
- 聯邦學習（FedAvg）
- MLOps / 模型版本控制
- Transformer / Self-Attention
- 對比學習（Contrastive Learning）

### 題目特點
- 題目長度適中（40~50 字）
- 選項含易混淆概念（需深度理解才能區分）
- 知識點均勻覆蓋，無章節偏重

## 六、常見錯誤與修復紀錄

### 1. 迴圈題目大量重複（v3.0 → v3.1 核心修復）
- **問題**：`for i in range(11,31): t=i%3` 中相同靜態文字被重複 6~8 次
- **處理**：刪除 `range(11,31)`/`range(1,31)` 迴圈，專業改為 10 個種子題，進階縮小為 `range(7,10)` + 補 1 題

### 2. `SyntaxError: closing parenthesis ']' does not match opening parenthesis '('`
- **檔案**：舊版 `gen_subject3_v3.py`（line 135）
- **原因**：`make_q()` 函數中括號不匹配
- **處理**：完整重寫，改用 explicit dict 格式（與科目一同結構）

### 3. `.split(" ")` 產生 8 元素選項
- **檔案**：舊版 `gen_subject3_v3.py`
- **原因**：`"A. 選項 B. 選項 ...".split(" ")` 在 Python 中產生 8 個元素
- **處理**：完整重寫，改用 explicit list：`["A. 選項", "B. 選項", ...]`

### 4. `KeyError: 'an'` / `KeyError: 'opts'`
- **檔案**：`gen_subject1_v3.py`、`gen_subject2_v3.py`、`gen_subject3_v3.py`
- **原因**：種子 dict 中誤用長格式 key name：
  - `"options"` 應為 `"opts"` • `"answer"` 應為 `"ans"` • `"key_point"` 應為 `"kp"` • `"analysis"` 應為 `"an"`
- **查找指令**：
  ```powershell
  grep -n '"analysis":"' gen_subjectX_v3.py   # 找誤用的key
  ```
- **處理**：將 seed dict 中全部改回短格式

### 5. `ValueError: Invalid format specifier`
- **檔案**：`gen_subject2_v3.py`（line 395, 680）
- **原因**：f-string 中的 `{}` 被誤認為格式標記
- **處理**：改為 `{{'age':[25,...]}}` 跳脫

### 6. 編碼問題（CP950/Big5 儲存）
- **檔案**：舊版生成的 `qbank_all_1080_v3_data.js`、`qbank_all_1170_v3_data.js`
- **原因**：Python 腳本用 `encoding='utf-8'` 寫入但內容已被 CP950 編碼
- **處理**：重新執行已修復的生成腳本，寫入正確的 UTF-8 檔案

## 七、注意事項

### 在 `f""` 字串中包含 `{}` 時
```python
# ❌ 錯誤：f-string 會嘗試解析 {}
q = f"data = pd.DataFrame({'age':[25,30]})"

# ✅ 正確：使用 {{}} 跳脫
q = f"data = pd.DataFrame({{'age':[25,30]}})"

# ✅ 或改用非 f-string
q = "data = pd.DataFrame({'age':[25,30]})"
```

### 種子題 key name 必須一致
```python
# ✅ 正確（短格式）
{"q":"...", "opts":[...], "ans":"A", "kp":"...", "an":"..."}

# ❌ 錯誤（混合長短格式）
{"q":"...", "options":[...], "ans":"A", "key_point":"...", "an":"..."}
```

### 專業 seed list 需補齊 `sXX_pro = [`
```python
# 注意 sXX_pro 行不能遺漏，否則 IndentationError
s61_pro = [
    {"q":"...","opts":[...],"ans":"A","kp":"...","an":"..."},
    # ... 10 題
]
```

### HTML 題數標示需要更新
若修改了範圍數/題數，需同步更新 `ipas_quiz.html` 中以下位置：
- `subjectNames`（科目名稱）
- `infoTexts`（提示列文字，含範圍數與總題數）
